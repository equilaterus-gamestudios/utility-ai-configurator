
# Geppetto - UtilityAI Configurator

![Geppetto UtilityAI Configurator](https://equilaterus.com/assets/img/projects/geppetto-utility-ai.PNG)

UtilityAI for Unreal by Equilaterus.

* Run: 
  ```
  npm start
  ```

* Build dist:
  ```bash
  npm run build-app
  # Build for linux
  npm run build-linux
  # Build for win
  npm run build-win
  ```
